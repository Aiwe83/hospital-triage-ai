# Changelog

Historial de versiones de este repositorio.
Formato basado en [Keep a Changelog](https://keepachangelog.com/es/).

**Convención de versionado:** cada commit nuevo crea un tag anotado `Vn`
(V1, V2, V3 …) y una entrada aquí describiendo sus cambios.
Categorías: `Added` / `Changed` / `Removed` / `Fixed` / `Security`.
